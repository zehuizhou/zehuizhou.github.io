assert

发表于 2018-10-23 | 分类于 pytest | 阅读次数

前言

断言是写自动化测试最重要的一步，一个用例没有断言，就失去了自动化测试的意义了。什么是断言呢？简单来讲就是实际结果和期望结果去对比，符合预期那就测试pass，不符合预期那就测试 failed

pytest允许您使用标准Python断言来验证Python测试中的期望和值。例如，你可以写下:

def f():
    return 3

def test_function():
    assert f() == 4

异常信息：

在异常的时候，输出一些提示信息，这样在报错后，就方便查看是什么原因

def f():
    return 3

def test_function():
    a = f()
    assert a % 2 == 0, "判断a为偶数，当前a的值为："%a

异常断言：

import pytest

def test_zero_division():
    '''断言异常'''
    with pytest.raises(ZeroDivisionError) as excinfo:
        1 / 0
    # 断言异常type
    assert excinfo.type == ZeroDivisionError
    # 断言异常value
    assert "division by zero" in str(excinfo.value)

注意：断言type不需要加引号，断言value值的时候需要转str

常用断言

assert xx
assert not xx
assert a in b
assert a == b
assert a !=b
