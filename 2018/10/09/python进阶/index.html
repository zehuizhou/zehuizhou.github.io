一切皆对象

类：

对象属性定义在方法内部
类属性定义在类内部，方法外部

self：方法内部引用对象自身
1.操作对象属性
2.一个方法内部调用同一类的其他方法

init()方法：python会在每次创建对象时自动调用，因此，我们可以在init()方法内部来初始化对象属性

class Bird():
    def __init__(self, sound):
        self.sound = sound
        print("my sound is", sound)
    def chir(self):
        print(self.sound)

summer = Bird("dudu")
summer.chir()

闭包 ：一个函数和他的环境变量合在一起，就构成了一个闭包

1.提高代码的可复用性
2.减少函数参数 

def line_conf():
b = 15
    def line(x):
    return 2*x + b
b = 5
return line

if __name__=="__main__":
my_line = line_conf()
print(my_line.__closure__)
print(my_line.__closure__[0].cell_contents)
print(my_line(5))
打印出15，return line之前,b=5

装饰器 ：可以对一个函数、方法或者类进行加工，定义功能拓展本身

例子：把time包来测量程序运行时间的功能做成一个装饰器，这个装饰器可以运行于其他函数，将显示函数的运行时间

import time

def decorator_timer(old_function):
    def new_function(*arg, **dict_arg):
        t1 = time.time()
        result = old_function(*arg, **dict_arg)
        t2 = time.time()
        print("time:", t2-t1)
        return result
return new_function

@decorator_timer
def hello_world():
    print("hello world")

if __name__ == "__main__":
    hello_world()
