shell实战演练

发表于 2016-08-31 | 分类于 shell | 阅读次数

先写个非常非常简单的脚本
打开终端，依次输入以下语句：
echo ‘#!/bin/bash’ > test.sh
echo ‘echo “hello world”‘ >> test.sh
这样一个脚本就写好了，
sh test.sh
没出错的话应该输出
hello world

接下来进行实战演练，先说需求，以root用户登录，为每个sd硬盘在桌面创建一个文件夹，文件夹以sd的形式命名，文件夹中包含一个文件，命名是date命令获取的时间
tips: date的命令为”date +%Y.%m.%d_%H:%M:%S”
文件中保存着硬盘的大小值

#!/bin/bash
disk_detail_list=$(fdisk -l 2> /dev/null | grep -i "disk /dev/sd")
disk_name_list=`echo $disk_detail_list | awk '{print $2}' | awk -F/ '{print $3}'| tr -d [:]`
disk_size_list=$(echo $disk_detail_list| awk '{print $3" "$4}')
disk_num=$(echo $disk_detail_list|wc -l)
for ((i=1;$i<=$disk_num;i++))
do
    #disk_name=$(echo $disk_name_list| sed -n -e "${i}p")
    disk_name=$(echo $disk_name_list| head -n $i | tail -n 1)
    #disk_size=$(echo $disk_size_list| awk -F, '{if (NR == "'"$i"'") print $1}')
    disk_size=$(echo $disk_size_list| awk -F, "{if (NR == $i) print \$1}")
    mkdir /root/Desktop/$disk_name
    file_name=$(date +%Y.%m.%d_%H:%M:%S)
    echo "$disk_size" > /root/Desktop/$disk_name/$file_name
done
# shell
