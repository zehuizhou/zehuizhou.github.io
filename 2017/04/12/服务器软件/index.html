服务器软件

发表于 2017-04-12 | 分类于 服务器 | 阅读次数

BIOS

  1. Basic Input Output System 电脑中最基本也是最重要的输入输出程序。里面装有系统的重要信息和设置系统参数的程序。存放在主板上的一块不需要电源供电的EPROM芯片上。
  2. BIOS的功能
    通电自检功能 ：检查基本硬件设备是否正常
    系统初启时的系统设置、初始化程序：对硬件设备初始化，为工作做初始的准备。
    BIOS设置：对硬件进行相关的设定。
    启动系统：根据相关启动项进入系统。

BMC

  1. Baseboard Management Controller外界可以通过BMC模块按照IPMI协议获取该单板上的信息，如电源功率、风扇、传感器、软件版本等。
  2. IPMI智能平台管理接口

系统

  1. 同常会根据需求装，如：Centos、Redhat、Suse、Windows2012 R2等
  2. windows系统安装过程中需要打驱动。
